Paso 1 Librerias que debemos importar y su significado

1. Pandas (pd)

Propósito: Pandas es una biblioteca fundamental para la manipulación y análisis de datos en Python.

Uso en el código: Cargamos el archivo CSV en un DataFrame con pd.read_csv(). Seleccionamos columnas relevantes, renombramos columnas, y limpiamos los datos. Pandas también permite realizar operaciones como conversión de tipos de datos, cálculo de estadísticas descriptivas, y manejo de valores nulos.

Funciones importantes de Pandas:

  • pd.read_csv(): Carga datos desde un archivo CSV.
  • .copy(): Crea una copia del DataFrame para manipular los datos sin afectar el original.
  • .replace(): Reemplaza valores no deseados, como '#DIV/0!' y '', con valores NaN.
  • .to_numeric(): Convierte las columnas a tipos de datos numéricos.
  • .dropna(): Elimina filas con valores nulos.
  • .fillna(): Llena valores nulos con algún valor, en este caso, la media de la columna.
  • .describe(): Proporciona estadísticas descriptivas como la media, desviación estándar, etc.

2. Numpy (np)

Propósito: Numpy es una biblioteca que ofrece soporte para arrays y operaciones matemáticas. Es la base de muchas otras bibliotecas de análisis de datos.

Uso en el código:

np.nan: Utilizado para representar valores faltantes o no disponibles (NaN: Not a Number).

Funciones importantes de Numpy:

  • np.nan: Se usa para asignar valores nulos en los datos.

3. Plotly Express (px)

Propósito: Plotly Express es una biblioteca para crear gráficos interactivos y de alta calidad. Permite generar gráficos con muy pocas líneas de código y con una personalización intuitiva.

Uso en el código:

Creamos gráficos interactivos para visualizar las relaciones entre variables (por ejemplo, PIB vs Población, Emisiones de Carbono vs Superficie Construida).

Funciones importantes de Plotly Express:

  • px.scatter(): Crea gráficos de dispersión, permitiendo personalizar variables como el tamaño de los puntos y los colores.
  • fig.show(): Muestra el gráfico en una salida interactiva.

4. Plotly Graph Objects (go)

Propósito: Es una extensión de Plotly que permite crear gráficos más personalizados y detallados. Ofrece un mayor control sobre el estilo y formato de los gráficos.

Uso en el código:

Creamos un heatmap (mapa de calor) de la matriz de correlación para visualizar las relaciones entre las variables de una manera visualmente atractiva.

Funciones importantes de Plotly Graph Objects:

  • go.Figure(): Crea un objeto de gráfico (en este caso, un heatmap).
  • go.Heatmap(): Específicamente utilizado para generar mapas de calor.
  • fig.update_layout(): Personaliza los detalles del gráfico como el título, los ejes, entre otros

5. Scipy Stats (zscore)

Propósito: Scipy es una biblioteca que proporciona herramientas estadísticas y científicas. El módulo stats se especializa en funciones estadísticas, como el cálculo de Z-scores.

Uso en el código:

Usamos zscore para calcular la puntuación estándar (Z-score) de cada columna, lo que nos ayuda a identificar valores atípicos (outliers). Funciones importantes de Scipy Stats:

  • zscore(): Calcula el Z-score de cada valor en una columna o conjunto de datos. Los Z-scores se utilizan para medir cuán lejos está un valor de la media en términos de desviaciones estándar. Valores con un Z-score mayor a 3 o menor a -3 suelen ser considerados valores atípicos (outliers).
